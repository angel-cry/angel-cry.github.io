发现个好东西Vercel,拥有全球CDN，国内速度不错,可以用来加速github的访问速度。

Vercel

如果你曾听说过 ZEIT Now，是的，它是 Vercel 的前身。在今年早些时候，ZEIT Now 进行了「品牌升级」，成为了现在的 Vercel。。
如果你的博客和我一样搭建在github上的，那么用Vercel来加速一下，是真的香！

使用

  • 打开 官网，你可以使用github账号来关联登录。

  • 在Overview选项页，点击【New Project】

  • 在Improt Git Repository 上添加要导入的项目，并点击Import。

  • 接着选择你的账户

  • 继续

  • 再点击Deploy,就开始构建了

  • 构建完成后你就可以通过vercel自动配置的子域名访问了。

设置域名

  • 点击你创建的项目，进入配置界面，点击Settings->Domains,输入想设置的域名，点击Add.

  • Add之后，可能还需要配置一下我们的域名解析

  • 打开域名管理器，添加一条CNAME记录。

  • 添加成功后，我们回来卡片就会变成如图的样子

现在你就可以通过设置的域名来访问了。需要更新网站的时候，只需要往github上提交新的commit,vercel就会自动重新构建，解放双手，值得拥有~
